Adding a DNS Record

  1. Edit infrastructure/services.yml
  2. Find or add host under managed_hosts_services
  3. Add to service_domains:
docker:
  service_domains:
    - domain: newapp.voidlabs.cc
      target: caddy   # Through reverse proxy
    - domain: direct.voidlabs.local
      target: host    # Direct to host IP
  1. Run: ansible-playbook playbooks/opnsense_dhcp.yml

Target Options

  • target: host ‚Üí Host's reserved_ip from hosts.yml
  • target: caddy ‚Üí Caddy proxy (192.168.51.80)
  • target: <hostname> ‚Üí Another host's IP
  • target_ip: x.x.x.x ‚Üí Literal IP override

Key IPs

  • Gateway: 192.168.51.1 (opnsense)
  • DNS: 192.168.51.53 (adguard)
  • Caddy: 192.168.51.80
