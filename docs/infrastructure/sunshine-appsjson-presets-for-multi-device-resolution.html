Sunshine Apps.json Presets for Multi-Device Resolution

Created: 2026-01-04 00:00:00
sunshine moonlight apps.json resolution ipad macbook hyprland headless

Overview

Sunshine's apps.json file (~/.config/sunshine/apps.json) defines application presets that appear in Moonlight clients. These can dynamically configure display resolution for different streaming devices.

apps.json Location

  • Linux: ~/.config/sunshine/apps.json
  • Edited via Sunshine web UI at https://<host>:47990 or directly

Multi-Device Resolution Presets

Use prep-cmd to dynamically resize the headless display based on the streaming client device.

‚ö†Ô∏è Important: Sunshine executes prep-cmd directly without a shell. You must wrap commands in bash -c '...' for shell features (pipes, &&, command substitution, export) to work.

Example apps.json with Device Presets

```json { "env": { "PATH": "$(PATH):$(HOME)/.local/bin" }, "apps": [ { "name": "MacBook Desktop", "image-path": "desktop.png", "prep-cmd": [ { "do": "bash -c 'export HYPRLAND_INSTANCE_SIGNATURE=$(ls /run/user/1000/hypr/) && hyprctl dispatch focusmonitor HEADLESS-1'", "undo": "bash -c 'export HYPRLAND_INSTANCE_SIGNATURE=$(ls /run/user/1000/hypr/) && hyprctl dispatch focusmonitor DP-1'" } ] }, { "name": "iPad Air 11 Desktop", "image-path": "desktop.png", "prep-cmd": [ { "do": "bash -c 'export HYPRLAND_INSTANCE_SIGNATURE=$(ls /run/user/1000/hypr/) && HEADLESS=$(hyprctl monitors -j | jq -r "[.[] | select(.name | startswith(\"HEADLESS\"))] | .[0].name") && hyprctl keyword monitor "$HEADLESS,2360x1640@60,auto,1" && hyprctl dispatch focusmonitor $HEADLESS'", "undo": "bash -c 'export HYPRLAND_INSTANCE_SIGNATURE=$(ls /run/user/1000/hypr/) && HEADLESS=$(hyprctl monitors -j | jq -r "[.[] | select(.name | startswith(\"HEADLESS\"))] | .[0].name") && hyprctl keyword monitor "$HEADLESS,2880x1864@60,auto,1.5" && hyprctl dispatch focusmonitor DP-1'" } ] }, { "name": "Desktop (DP-1)", "image-path": "desktop.png" }, { "name": "Steam Big Picture", "detached": ["setsid steam steam://open/bigpicture"], "prep-cmd": [ { "do": "", "undo": "setsid steam steam://close/bigpicture" } ], "image-path": "steam.png" } ] } ```

Key Concepts

prep-cmd Structure

  • do: Runs when connecting to this app from Moonlight
  • undo: Runs when disconnecting

Dynamic Headless Detection

The jq command finds the current headless monitor name: ```bash HEADLESS=$(hyprctl monitors -j | jq -r '[.[] | select(.name | startswith("HEADLESS"))] | .[0].name') ```

This handles the HEADLESS-N naming which increments during the session.

Hyprland Resolution Change

```bash hyprctl keyword monitor "$HEADLESS,2360x1640@60,auto,1"

name WxH@Hz pos scale

```

Apply Changes

After editing apps.json: ```bash systemctl --user restart sunshine ```
